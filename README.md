## 📖 Sobre

Este projeto foi desenvolvido como forma de colocar em prática os conceitos estudados durante o curso da Alura, explorando principalmente o uso do Spring Boot junto com recursos modernos da linguagem Java.

A aplicação utiliza diversas funcionalidades do ecossistema Java, como coleções, lambdas, Streams API, records e interfaces, com o objetivo de manipular e processar dados de séries e episódios de forma eficiente e legível.

Além do aprendizado teórico, o projeto representa uma adaptação prática e personalizada, onde foram feitas melhorias e implementações próprias para reforçar o domínio dos conceitos, exercitando boas práticas de programação e estruturação de código.

## ✨ Funcionalidades

- ✅ Busca por nome da série.
- ✅ Busca o nome do episódio e retorna a temporada que esse episódio esta localizado.
- ✅ Filtra episódios a partir do ano.
- ✅ Filtra temporadas por média de avaliação.
- ✅ Retorna todos os episódios para cada temporada.
- ✅ Retorna estatísticas da série por completo.
